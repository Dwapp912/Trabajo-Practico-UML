# Trabajo-Practico-UML
## Contenido del Repositorio

El proyecto implementa 14 ejercicios prácticos que cubren los siguientes tipos de relaciones:
* Asociación (Unidireccional y Bidireccional)
* Agregación
* Composición
* Dependencia (De Uso y De Creación)

## Estructura de Carpetas

Todo el código fuente se encuentra dentro de la carpeta `TP5_UML/`.

La ruta completa a los paquetes de Java es:
**`TP5_UML/trabajo/practicon5/`**

Dentro de esa carpeta, cada uno de los 14 ejercicios está separado en su propio paquete (ej: `ejercicio1`, `ejercicio2`, etc.).

## Ejecución

Cada paquete de ejercicio (ej. `ejercicio1`) es independiente y contiene su propia clase `Main.java` que puede ser ejecutada para probar la funcionalidad de ese ejercicio en particular.
